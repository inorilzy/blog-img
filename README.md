# blog-img

> 个人博客和图床笔记的图片仓库。

[中文](README.md) | [English](README.en.md)

## 用途

本仓库存储博主博客、笔记和文档中用到的静态图片，不是一个可复用的软件项目。

## 目录结构

```text
blog-img/
├── img/          # 分组图片资源
└── *.png / *.jpg # 兼容旧用的 root 层级图片
```

## 使用说明

图片通过 GitHub raw URL 或博客内容引用。不建议将此仓库当作有可用性保证的图片 CDN。

## 注意事项

- 不要删除或重命名已被外部引用的图片。
- 新图片建议放在 `img/` 下，而不是仓库根目录。
- 图片版权可能因来源而异，引用前请确认许可。
